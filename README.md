A comprehensive interactive guide designed to simplify the deployment and configuration of **DNSTT** (DNS Tunneling). This walkthrough provides a step-by-step path for users to establish secure connectivity through DNS protocols, specifically optimized for challenging network environments.

## üìñ What's Inside

This application serves as a complete manual for the DNSTT ecosystem, containing:

- **Guided Deployment:** Interactive instructions covering everything from initial VPS requirements to final server execution.
- **DNS Configuration Manual:** Detailed walkthrough for setting up A and NS records correctly to enable tunneling.
- **Visual Client Setup:** A dedicated section with high-resolution screenshots for configuring mobile applications like HTTP Injector on Android and iOS.
- **Technical Glossary:** Integrated explanations for networking terms (DNS, MTU, SSH, etc.) to help users understand the underlying technology.
- **Server Maintenance Suite:** A collection of essential commands for managing users, monitoring logs, and updating the server post-installation.
- **Bilingual Interface:** Fully localized content in both **English** and **Persian (Farsi)**, featuring a responsive Right-to-Left (RTL) design for native reading experiences.
